## 6. 关键认知 / 避撞 / 运维（综合速查）

- **多窗共享 git index**：别窗 `git add` 的文件会被你的 commit 捎走。**commit 前必 `git diff --cached --stat` 核对暂存区**。
- **重启生产** `launchctl kickstart -k gui/$(id -u)/com.noe.panel`（不要 npm start）；**改 .env 必 kickstart**（dotenv 启动读一次）。
- **测语音 API 必带**：文本含唤醒词（owner-gate.json enabled=true，自定义唤醒词「宝贝/hi宝贝/hi」）+ `voice:true` 才出声。没带=静默 ignored，别误诊成"模型挂了"。
- **LM Studio JIT**：切模型首请求 7-180s 属正常加载；连续两次空响应≠挂。本地大脑请求必走回环不经代理（NO_PROXY 护 127.0.0.1:1234）。
