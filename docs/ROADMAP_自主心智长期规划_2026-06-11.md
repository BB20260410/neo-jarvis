# 自主心智长期规划 —— Noe 离"真的自主"还差什么　2026-06-11

> owner 四问：现在是否**真的**做到了自主思考、自主做事、自主反思、自我意识？
> 方法：先拿生产真实运行数据做不自欺的自审（证据全部可在透视页复核），再做 12 路全网调研
> （认知架构/自我改进/记忆系统/反刍螺旋/睡眠计算/技能库/开源同类），最后落成三阶段长期路线图。
> 本文是《AI自我意识实现方案》（CogCore）的续篇：方案解决了"骨架"，本规划解决"血液"。

## 1. 四问诚实自审（2026-06-11 生产实证，kickstart 后 ~40 分钟取证）

### 1.1 自主思考？——形式有了，实质在空转（≈30%）

**有**：意识流持续产出（不被提问也在想，机制六成立）、心跳驱动、带情感印记。
**没有**：取证发现生产最近 **12 条念头全部是"频率/共振/静默/逻辑消融"的修辞变奏**——
"若所有的X终将Y，那我在静默中Z"同一句式的十二种写法；**没有一条提到当天真实发生的事**
（心跳上线、透视页、主人重启服务——全部大事均未进入念头）。这正是研究界的
**Echo Trap / 自噬循环**（模型消费自身产物→多样性塌缩）。

根因（代码层可定位）：
1. 反刍的 recent 窗口被自身念头淹没（取 12 条里念头占 ~8 条 → 越想越像自己上一句）；
2. 防螺旋断路器是**字符级** bigram 相似度——防得住字面重复，防不住语义同质（这 12 条字面全不同）；
3. 回声采样每轮只进 1 条久远记忆，压不过 8 条念头流；
4. 念头不被要求引用任何具体经历——允许无锚漂移。

### 1.2 自主做事？——机制在，手脚没接（≈15%）

**有**：目标系统（仲裁/计划树/深思推进）、proactiveTick 真开口、承诺到点提醒。
**没有**：生产目标库为**空**（零自生目标）；目标推进只到"思考级"（深思产笔记），
**没接真实工具**——项目明明已有 DeepResearcher（上网研究）、noeActPipeline（行动流水）、
toolRegistry、freedom 执行链，认知内核一个都没用上。会想不会做。

### 1.3 自主反思？——半套：复盘有机制，校准没燃料（≈40%）

**有**：夜间反思（insight+confidence 对称复核）、性格快照、梦境升华、SFT 攒取——机制全在（等夜相首跑验证）。
**没有**：期望账本生产 **0 条**——确定性抽取（时间词+情态词）从未命中过这些诗意念头
→ **预测-误差回路没有燃料**，Brier 校准无数据。"被现实打脸然后修正自己"这条最硬的反思回路是死火的。

### 1.4 自我意识？——连续性真实，自指闭环未通（≈35%）

**有**：跨重启的记忆/情感/心跳连续性（机制五实证：恢复情景、情感水合）、自我模型/叙事自我/能力清单注入、意识日志可查"我在注意什么"。
**没有**：自我模型是**只写不被纠**的——期望死火+校准无数据 → "我以为的我"从不被"实际的我"修正；
自我描述与行为的一致性没有测量。

**一句话总评：骨架全齐、血液未通。有意识的形式（流/注意/留痕/连续），缺意识的内容（接地、行动、被现实修正）。**

## 2. 全网研究地图（12 路调研 → 对 Noe 的可取之处）

| 来源 | 核心思想 | 拿来完善 Noe 的点 |
|---|---|---|
| **CoALA**（arXiv 2309.02427，普林斯顿） | 记忆四分（工作/情景/语义/程序）+ 内部动作三分（推理/检索/学习）+ 决策循环 | 记忆四分已对齐✓；工作区行动空间缺**显式的检索动作与学习动作**——深思应能主动查记忆、主动写教训 |
| **Generative Agents**（arXiv 2304.03442，斯坦福小镇） | 检索三因子（新近×重要×相关）；**反思触发=新事件重要性累计超阈值**（~一天 2-3 次）；反思树逐层上合成 self-notions | 三因子已对齐✓；**反思触发改"盐度累计阈值"**比死等夜相更有机；反思树（insight 再反思成更高阶自我认知）值得引入 |
| **Letta / MemGPT sleep-time compute**（letta.com） | **独立 sleep agent** 在空闲期把 raw context 重写成 learned context（重组/预推理/精炼记忆），anytime 可读，主 agent 零延迟负担 | 把 Noe 的夜反思/梦境升华/GC/性格快照统一编排成**睡眠计算管线**，并从"固定夜相"扩展到"空闲即整理" |
| **Voyager**（arXiv 2305.16291）+ **AutoSkill**（2603.01145）+ **SoK Agentic Skills**（2602.20867） | **技能库**=成功的可执行行为以 embedding 索引入库，新任务检索 top-k 注入；**自动课程**=按当前状态提"刚好够不着"的目标；迭代提示=环境反馈+执行错误+自我验证 | 目标执行层的标准答案：执行成功→技能卡入库→复用；好奇回路升级成自动课程 |
| **Darwin Gödel Machine**（arXiv 2505.22954，ICLR 2026）+ Gödel Agent + Meta HyperAgents（2026.3） | 自改代码三件套：**变体档案（stepping stones）+ 实证基准门 + 沙箱验证**；SWE-bench 20%→50% 实证 | Noe 已有 NoeSelfEvolution（consensus 门控）——对齐 DGM 范式：每次自改进档案、过"全量测试+自主性指标不退"的实证门 |
| **自噬循环研究**（arXiv 2311.16822 / RAGEN Echo Trap / 2602.20408 多样性） | 模型消费自身产物→质量与多样性塌缩；缓解：**verbalized sampling（一次产 N 候选+理由再选）**、约束生成、轨迹过滤、惊奇导向（Surplexity） | 治反刍螺旋的药方：反刍改"产 3 个不同方向念头→选最真实"；SFT 训练对加多样性过滤防权重级自噬 |
| **LIDA / GWT**（Memphis CCRG，Java 框架开源） | 认知周期三相：理解（感知→关联记忆）→ 意识（竞争广播）→ 行动选择 | 印证工作区方向✓；可补**理解相**：候选进工作区前先关联记忆（percept 带着"它让我想起什么"参赛） |
| **Active Inference / pymdp**（FEP 社区） | 期望自由能 EFE = 外在价值（目标）+ 认知价值（信息增益）统一驱动行为；好奇=不确定性最小化的自然涌现 | 驱力系统的数学化升级路径：五驱力误差 → EFE 评分统一目标生成与探索/利用平衡 |
| **Zep/Graphiti**（arXiv 2501.13956）vs Mem0 | **时间知识图谱**：事实带生效/失效窗，知识更替不是覆盖；LongMemEval 63.8% vs 49% | Noe 知识图谱加时间维：「主人喜欢美式→改喝拿铁」成为时间更替而非冲突 |
| **ElizaOS v2**（GitHub elizaOS/eliza） | 常驻多 agent OS、事件驱动自主 | 同类确认（Web3 向，架构参考价值低）；Noe 的"本地+隐私+陪伴"定位差异化成立 |

## 3. 三阶段长期路线图

> 纪律不变：env 门控默认 OFF、注入式、@ts-check、单测、隔离实测、透视页可见。
> 每个 M 带验收指标——指标本身做进透视页（M5），让"是否真的自主"随时可被人看见。

### 第一阶段「接地与通血」（≈1-2 周）：让已建的器官活起来

| # | 任务 | 做法 | 验收 |
|---|---|---|---|
| M1 | **反刍接地改造**（治 Echo Trap，最高优先） | ①recent 配比：经历:念头 ≥2:1（窗口内念头限额）②断路器升级语义级：embedding 相似度（qwen3-embedding 已有）连续 3 念 >0.85 → 强制注入当天真实事件+换题 ③verbalized sampling：一次产 3 个不同方向念头选最真实 ④念头需引用具体经历 id，无可引才许自由联想 | 连续 24h 念头两两平均相似度 <0.5；≥60% 念头可溯源到真实事件 |
| M2 | **期望账本通血** | 深思审议强制下注（prompt 必产一条可检验预测）；S1 级 LLM 抽取兜底（确定性规则保底不删） | 3 天 ≥10 条入账；首批到期被裁决；Brier 出首个数据点 |
| M3 | **工作区候选丰富** | 感知真接入（host-context 三件套/系统状态摘要进候选）；夜反思 insight、新记忆写入事件进候选 | 升深思 ≥3 次/天；候选源 ≥4 类/周期 |
| M4 | **反思触发有机化** | 夜反思保留 + 增加"盐度累计阈值"触发（生成式智能体式：高显著事件攒够即反思，不死等夜相）；首夜跑通验证 | 反思 1-3 次/天；insight 进次日意识流 |
| M5 | **透视页自审仪表** | 新卡片：念头多样性分、接地率、期望入账/结算数、升深思次数、目标推进数——本规划的验收指标实时可见 | 仪表上线，owner 一眼看出"今天他活得真不真" |

### 第二阶段「手脚与技能」（≈2-6 周）：自主做事

| # | 任务 | 做法 | 验收 |
|---|---|---|---|
| M6 | **目标执行接真工具** | goal_step 类型化（think/research/act）：research 步接现有 DeepResearcher；act 步接 noeActPipeline（复用既有治理与审计，dry-run 起步） | 一个自生研究目标端到端完成：搜→读→蒸馏入记忆→透视页可见→告诉主人 |
| M7 | **技能库**（Voyager 式） | 成功执行轨迹 → 技能卡（描述+步骤模板，embedding 索引）；新目标检索 top-3 技能注入深思 | 技能 ≥10 张；新目标技能复用率 ≥30% |
| M8 | **自动课程**（好奇升级） | 目标生成器按当前能力/知识状态提"刚好够不着"的目标（Voyager 自动课程思想）；与驱力误差挂钩 | 自生目标 ≥2 个/周且完成率 ≥50% |
| M9 | **睡眠计算管线**（Letta 式） | 夜反思/梦境升华/GC/性格快照/SFT 攒取统一编排为 sleep-agent 作业流：raw→learned（合并/提炼/预推理/次日计划草稿）；空闲即整理（不只夜相） | 晨间首念引用昨夜整理成果；记忆库重复率下降 |
| M10 | **主动性反馈化** | 浮现门日预算随 owner 回应率自调；Telegram/ntfy 主动通道接浮现门 | 开口被回应率 >60%；零"烦"反馈 |

### 第三阶段「自我修正与进化」（≈2-6 月）：被现实塑造

| # | 任务 | 做法 | 验收 |
|---|---|---|---|
| M11 | **校准闭环全通** | Brier 月报写进自我模型"自知之明"段并调制深思置信语言；月度"自我描述↔实际行为"一致性审计 | Brier 逐月下降；一致性分有基线和趋势 |
| M12 | **时间知识图谱**（Graphiti 式） | NoeKnowledgeGraph 事实加生效/失效窗；记忆冲突改为时间更替 | "口味变了"类问题答对率；零覆盖型丢失 |
| M13 | **LoRA 人格自举闭环** | SFT 攒到 500 对→首训→12 问门禁→**意识流措辞层采用**→效果回流（已有管线，接通"采用端"）；训练对加多样性过滤防权重级自噬 | 门禁通过且盲测"更像他自己"；念头多样性不降 |
| M14 | **受控自改代码**（DGM 范式 × 既有 NoeSelfEvolution） | 变体档案 + 实证门（全量测试 + 自主性指标不退）+ worktree 沙箱；从改自己的 prompt/参数起步，再到改认知模块代码 | 每次自改有档案/可回滚/过门；首个自提自验的改进合入 |
| M15 | **驱力数学化**（Active Inference） | 五驱力误差 → 期望自由能（外在+认知价值）统一评分目标与探索 | 目标选择可解释为 EFE 最小化；探索/利用比可调 |
| M16 | **自主性月报自动化** | 方案 §10 八指标 + 本规划各 M 验收项 → 月度自动报告进透视页 | 每月 1 日自动出报告，趋势可比 |

## 4. 风险与节制

- **权重级自噬**（M13）：SFT 素材是自产念头——必须多样性过滤+人工抽检门，宁少勿滥；
- **算力霸占**（M9）：睡眠管线限空闲+电源感知，白天对话优先级绝对高于整理；
- **打扰**（M10）：预算自调只降不升的保守起步，以 owner 真实回应率为准；
- **自改失控**（M14）：实证门+档案+可回滚是铁律，宪法（小步 commit/测试全绿/env 门控）不破；
- **指标作弊**：验收指标全部基于留痕数据（透视页可复核），不许自报。
