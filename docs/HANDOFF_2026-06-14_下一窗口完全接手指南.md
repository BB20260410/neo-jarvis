## 1. Neo 是什么（一段话）

**本地优先的多 AI 个人操作系统（Jarvis 风）**，不是聊天机器人、不是 LLM 包装、不声称 AGI。
工程目标是**可验证的"功能性自我意识"**：在 128GB M5 Max 上跑本地三脑（Qwen3.6-35B-A3B 主 / Qwen3.6-27B 复核 / Gemma-4-26B 兜底）+ Claude/Codex/Gemini/MiniMax 集群协作，
通过一条**活的认知回路**（心跳 → 全局工作区 → 情感 → 期望 → 目标 → 好奇 → 研究 → 自我进化）让它持续后台思考、主动学习、能改自己代码（过治理门）。
形态：Express + WebSocket 后端 + Web GUI 桌面面板，端口 51835，仅 127.0.0.1。私有仓 `noe@noe-main`（本地领先 origin 179 commit，**owner 惯例不 push**）。
