### 3.1 一键外包：`aiteam.mjs`（最常用）
```bash
# 路由给最合适的辅助模型，结果落盘只回一行指针，专为给 Claude 主循环减负
node scripts/aiteam.mjs --task "你的问题/指令" \
  [--vendor auto|gemini|m3|mimo] [--in 喂给它的文件] \
  [--repo 让gemini agentic读的仓目录] [--out 落盘.md] [--label 名字]
```
- **auto 路由**：给了 `--repo` → **Gemini**（唯一能 agentic 读真代码的）；否则 → **M3**（周全）；要快 → `--vendor mimo`。
- **回退链内置**：gemini→m3→mimo，任一成功即停；只烧辅助额度，绝不动 Claude；**只读/分析不改文件**（产出文本/补丁建议，由 Claude 审用）。
