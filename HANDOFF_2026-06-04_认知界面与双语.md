- **入口**：主面板 sidebar「⚙️ 设置」→ 工作台 → 🪐 认知；或直接 `/cognitive.html`（token 复用 `localStorage['panel-owner-token']`，从主面板跳转免带 ?t=）。
- **三栏**（复刻 BaiLongma Cognitive Surface）：左 L1 感知（实时视频自预览 getUserMedia + 每 4s 推帧）/ 中 认知图 + 聊天气泡（YOU 琥珀 / NOE 青）/ 右 L2 推理流。
- **认知图三风格**（顶部 ◑ 按钮循环，`localStorage['cog_theme']` 持久化）：暖核神经网 / 冷蓝全息 HUD / 深空星云。共用 canvas 引擎：力导向 + 数据流粒子 + 辉光呼吸 + 激活波纹 + 神经冲动传导，DPR=2。

- **实时语音**：能量 VAD 免按钮 + barge-in 打断 + 麦克风电平条；文字聊天（需重启后端 `chatText`）。

### 3. 全量中英双语
- `public/i18n.js`（新建）：**运行时翻译引擎**，零侵入 app.js 13k 行——TreeWalker 翻中文文本节点 + MutationObserver 捕获动态内容 + 整句&子串双匹配 + 存原文（`node.__o`）无损切回。
- `public/i18n-dict.json`（新建，1041 条）：sonnet agent 批量翻译。**扩字典只需往这个 JSON 加 `"中文":"English"`**，刷新即生效（i18n.js boot 时 fetch 加载 + 重算 SUB_KEYS）。
- 左下角 **🌐 中/EN** 一键切换（`localStorage['noe-lang']`）。
- 已验证翻全：欢迎页 / sidebar / 状态栏 / inspector / Brain / 房间 / 治理（治理 100%）。

### 4. 实时语音诊断（两个界面）
- **根因定位**：后端 curl 实测全链路正常 —— whisper STT(:8123) ✓ / 大脑 ollama `huihui_ai/qwen3.5-abliterated:4b` ✓ / MiniMax TTS ✓。问题在前端麦克风/VAD（戴耳机后系统麦克风可能切到没声音的耳机麦）。

## 踩坑记录（避免重犯）
- i18n 初始 `walk(document.body)` 同步遍历主面板巨大 DOM **卡死主线程 >10s 整页崩** → 改 `walkAsyncBody`（TreeWalker 边走边译每帧 8ms + rAF）。
- 设 `document.documentElement.lang='en'` **误触发 Chrome 自动翻译**把英文 Noe→诺埃/Claude→克劳德反翻 → 删掉 + boot 设 `translate=no`。
- agent 写 JSON 时英文值里的双引号未转义会坏 JSON → 逐行正则 `fix()` 修复。
- 收集未翻中文要**切 zh 模式**（en 模式收到的是子串半翻译态，非原文）。
