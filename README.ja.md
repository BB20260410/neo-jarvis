## 🌌 なぜ Neo か

今日の AI は賢い。しかし**忘れっぽく、受け身で、他人のサーバーの上に住んでいる**。毎日あなたは自己紹介をやり直し、AI は答えた瞬間に忘れる。蓋を閉じれば、また他人同士だ。

**Neo は別の存在になろうとしている。**

あなたを覚え、画面を見て、話を聞く相棒。ひとつのモデルが単独で戦うのではなく、**複数の** AI モデルがあなたと肩を並べて働くワークスペース。そしてあなたが休んでいる間に、**自ら省み、自ら改善し、自分をより良くしていく**システム。

そして、それは**あなただけのもの**。あなた自身のマシン上で動き、データも記憶も思考もすべてローカルに留まる。これはクラウド製品のクライアントではない。これは**あなたの** AI OS だ。

## ✨ Neo にできること

> 🔄 **自ら進化する** — 自身の状態を省み、改善の方向を自ら定め、**自分のソースコードを書き換え**、テストを走らせ、ダブルグリーンゲートと複数モデルのレビューを通過して初めて反映する。AI は静止したツールではなく、成長するシステムになる。

> 🧩 **1つではなく、AI のチーム** — ローカルの主脳 + レビュー脳 + クラウドの強力なモデルがタスクごとに分担し、相互検証する。あるモデルの幻覚を、別のモデルがその場で捕らえる。

> 💾 **決して忘れない** — 3層の永続記憶(セマンティック知識ベース + ファイル記憶 + 記憶グラフ)。セッションを跨いであなたを覚え、学んだ教訓を次の対話に自ら持ち込む。使うほどあなたを理解する。

> 🎙️👁️🤝 **聞き、見て、自ら声をかける** — ローカルの音声入出力、画面を読むビジョンモデル、そして*節度ある*能動性 — スパムせず、適切なタイミングで話しかける。

> 🪞 **透明な内面** — 意識の流れ、目標、期待、感情状態(VAD / グローバルワークスペース)をすべて可視化。ブラックボックスではなく、*それが何を考えているか見える*。

## 🔬 「自己進化」は本当に動いている

```
   自身のデータを省みる(どこを良くできる?)
            │
            ▼
   自律的に方向を提案 ──► 価値アンカー:技術的な着地点がある?検証可能?スコア稼ぎではない?
            │
            ▼
   モデルが自分のソースを書き換え(小さく本物の論理改善を優先)
            │
            ▼
   ダブルグリーンゲート:変更前後ともテスト全緑  ──►  失敗ならロールバック
            │
            ▼
   複数モデルのレビュー ──► 通過で反映・記録 / 失敗なら正直に記録
```

価値アンカーと reward-hacking のサーキットブレーカーが、「進化しているように見せる」ための無意味な変更を防ぐ。これはロードマップ上の構想ではなく、**今動いている**仕組みだ。

## 🚀 クイックスタート

**Node.js 22.x が必要** — `npm start` はバージョンガード(`scripts/ensure-node22.mjs`)を通り、「22 以上」ではなく Node 22 系を厳密に要求します。デフォルトの `node` が別のメジャーバージョンの場合は Node 22 をインストールする(例: `nvm install 22`)か、`NOE_NODE_BIN` で Node 22 のバイナリを指定してください。

```bash
# 1) 依存関係のインストール
npm install --omit=dev   # ランタイムのみ — node_modules 約 210 MB、パネルの起動には十分
# npm install            # フルインストール(開発 / テスト実行用)— 約 850 MB+(Electron、Playwright、Vitest 等を含む)

# 2) 起動(デフォルト 127.0.0.1:51835、ローカルのみ)
npm start

# 3) 起動ログに表示される URL を開く — owner トークン付き:
#    🚀 Noe @ http://127.0.0.1:51835/?t=<owner-token>
```

> **重要:** ログに表示される **`?t=...` 付きの完全な URL** を開いてください。`http://127.0.0.1:51835` を直接開くとページの骨格は表示されますが、すべての API が 401 を返します — `?t=` のトークンが owner 認証そのものです(フロントエンドが `sessionStorage` に保存)。macOS では対話ターミナルからの `npm start` で正しい URL が自動的にブラウザで開きます。`.env` は任意 — 主なスイッチは [`.env.example`](.env.example) を参照。

## 🧭 初回起動で見えるもの

- **UI は現在中国語が主体です。** 英語 UI はロードマップ上にあり、当面はコード・ログ・README が英語の入口です。
- **ローカルモデルもクラウドキーもない場合、** パネルは起動しページも見られますが、チャットの応答は失敗します — 少なくとも 1 つの「脳」が必要です。最速の一歩: [LM Studio](https://lmstudio.ai)(デフォルト `http://127.0.0.1:1234/v1`)か [Ollama](https://ollama.com)(デフォルト `http://127.0.0.1:11434`)でローカルモデルを動かし、`cp .env.example .env` して `NOE_LMSTUDIO_URL` / `NOE_OLLAMA_URL` を向けてください。
- **音声・視覚は伴走ローカルサービスが必要**(Whisper STT、OpenAI 互換エンドポイントの VLM)。**自己進化はデフォルト OFF**(`NOE_SELF_EVOLUTION=1` で有効化、`NOE_SELF_EVOLUTION_REAL_APPLY=1` を設定しない限り dry-run)。
